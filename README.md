# python-flask-ca2-quizapp

# TECHNOLOGIES USED:
A)	BACKEND: PYTHON
B)	FRONTEND: HTML, CSS, JS
C)	DATABASE: MYSQL
D)	FRAMEWORK: FLASK

## PROPOSED SYSTEM
# A)	AUTHENTICATION:
1)	LOGIN
2)	REGISTER
3)	FORGOT PASSWORD
4)	CHANGE PASSWORD
5)	LOGOUT

# B)	TEACHERS:
1)	CREATE TEST
2)	VIEW QUESTIONS
3)	UPDATE QUESTIONS
4)	DELETE QUESTIONS
5)	TESTS HISTORY
6)	STUDENTS RESULTS

# C)	STUDENTS:
1)	GIVE TEST
2)	RESULTS
